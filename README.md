# Network-on-Chip-Based-Smart-Home-Security-System-Implementation-Using-FPGA
This project implements a Network-on-Chip (NoC) smart home security system on a Xilinx Artix-7 FPGA using Verilog. It integrates IR, gas, and float sensors as individual nodes via a mesh topology for parallel, low-latency processing. Alerts are sent via a SIM800L GSM module.
